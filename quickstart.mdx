## Prerequisites

You need Ruby installed on your system. If you haven't done that yet, see the [Installation guide](/installation).

Verify your installation before continuing:

```bash
ruby --version
# ruby 3.x.x (2024-xx-xx revision ...) [x86_64-linux]
```

## Your first Ruby program

<Steps>
  <Step title="Create a file">
    Create a file named `hello.rb` with the following content:

    ```ruby hello.rb
    puts "Hello, World!"
    ```
  </Step>
  <Step title="Run it">
    Execute the file with the `ruby` command:

    ```bash
    ruby hello.rb
    ```
  </Step>
  <Step title="See the output">
    You should see:

    ```
    Hello, World!
    ```
  </Step>
</Steps>

## Try Ruby interactively

`irb` (Interactive Ruby) lets you evaluate Ruby expressions immediately without creating a file. It is the fastest way to experiment with the language.

```bash
irb
```

```ruby
irb(main):001> 1 + 2
=> 3
irb(main):002> "hello".upcase
=> "HELLO"
irb(main):003> [1, 2, 3].map { |n| n * 2 }
=> [2, 4, 6]
```

Type `exit` or press `Ctrl+D` to quit.

<Tip>
  `irb` supports tab completion and multi-line input. It is useful for checking method names, testing small snippets, and exploring the standard library.
</Tip>

## Core language concepts

### Strings

Strings in Ruby can be delimited with single or double quotes. Double-quoted strings support interpolation with `#{}`.

```ruby
name = "Alice"
puts "Hello, #{name}!"        # => Hello, Alice!
puts 'Hello, #{name}!'        # => Hello, #{name}!  (no interpolation)

puts "ruby".length            # => 4
puts "ruby".upcase            # => RUBY
puts "  trim me  ".strip      # => trim me
puts "ha" * 3                 # => hahaha
```

### Numbers

```ruby
puts 10 + 3     # => 13
puts 10 - 3     # => 7
puts 10 * 3     # => 30
puts 10 / 3     # => 3   (integer division)
puts 10.0 / 3   # => 3.3333333333333335
puts 10 % 3     # => 1   (modulo)
puts 2 ** 10    # => 1024

puts 42.class        # => Integer
puts 3.14.class      # => Float
puts 42.to_f         # => 42.0
puts 3.14.to_i       # => 3
```

### Arrays

Arrays are ordered, indexed collections of any objects.

```ruby
fruits = ["apple", "banana", "cherry"]

puts fruits[0]        # => apple
puts fruits[-1]       # => cherry
puts fruits.length    # => 3

fruits.push("date")
puts fruits.inspect   # => ["apple", "banana", "cherry", "date"]

fruits.each do |fruit|
  puts fruit
end

doubled = [1, 2, 3].map { |n| n * 2 }
puts doubled.inspect  # => [2, 4, 6]

evens = [1, 2, 3, 4, 5].select { |n| n.even? }
puts evens.inspect    # => [2, 4]
```

### Hashes

Hashes store key-value pairs. Symbol keys (`:name`) are the most common convention.

```ruby
person = { name: "Alice", age: 30, city: "London" }

puts person[:name]    # => Alice
puts person[:age]     # => 30

person[:email] = "alice@example.com"

person.each do |key, value|
  puts "#{key}: #{value}"
end
# => name: Alice
# => age: 30
# => city: London
# => email: alice@example.com

puts person.keys.inspect    # => [:name, :age, :city, :email]
puts person.values.inspect  # => ["Alice", 30, "London", "alice@example.com"]
```

### Blocks and iterators

Blocks are anonymous chunks of code passed to methods using `do...end` or `{}`. They are central to Ruby's iterator pattern.

```ruby
# do...end style (common for multi-line blocks)
3.times do |i|
  puts "iteration #{i}"
end
# => iteration 0
# => iteration 1
# => iteration 2

# Brace style (common for single-line blocks)
[1, 2, 3].each { |n| puts n }

# Transforming with map
squares = (1..5).map { |n| n ** 2 }
puts squares.inspect  # => [1, 4, 9, 16, 25]

# Accumulating with reduce
sum = [1, 2, 3, 4, 5].reduce(0) { |total, n| total + n }
puts sum  # => 15
```

### Classes

Ruby is fully object-oriented. You define behavior with classes, and all instances share the class's methods.

```ruby
class Animal
  def initialize(name, sound)
    @name  = name
    @sound = sound
  end

  def speak
    "#{@name} says #{@sound}!"
  end
end

class Dog < Animal
  def initialize(name)
    super(name, "woof")
  end

  def fetch(item)
    "#{@name} fetches the #{item}."
  end
end

dog = Dog.new("Rex")
puts dog.speak          # => Rex says woof!
puts dog.fetch("ball")  # => Rex fetches the ball.
```

### Exception handling

Use `begin`/`rescue`/`ensure` to handle errors gracefully.

```ruby
begin
  result = 10 / 0
rescue ZeroDivisionError => e
  puts "Caught an error: #{e.message}"
ensure
  puts "This always runs."
end
# => Caught an error: divided by 0
# => This always runs.
```

## Running a Ruby script with arguments

Ruby exposes command-line arguments through the `ARGV` array:

```ruby script.rb
name = ARGV[0] || "World"
puts "Hello, #{name}!"
```

```bash
ruby script.rb Alice
# => Hello, Alice!
```
