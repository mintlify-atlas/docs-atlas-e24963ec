## Get started in seconds

<Steps>
  <Step title="Install Ruby">
    Use your platform's recommended method — rbenv, RVM, or the official installer.
    ```bash
    # macOS with Homebrew
    brew install rbenv
    rbenv install 3.4.0
    rbenv global 3.4.0
    ```
  </Step>
  <Step title="Write your first program">
    Create a file `hello.rb` and write your first Ruby program:
    ```ruby hello.rb
    # Everything in Ruby is an object
    greeting = "Hello, World!"
    puts greeting.upcase        # => "HELLO, WORLD!"
    puts greeting.length        # => 13
    puts greeting.reverse       # => "!dlroW ,olleH"

    # Blocks and iterators
    3.times { |i| puts "Run #{i + 1}" }
    ```
  </Step>
  <Step title="Run it">
    Execute your program from the terminal:
    ```bash
    ruby hello.rb
    ```
  </Step>
  <Step title="Explore the docs">
    Dive into the [Language Reference](/language/syntax) to learn Ruby's syntax, or jump straight to [Core Classes](/core/string) to see what's available out of the box.
  </Step>
</Steps>
